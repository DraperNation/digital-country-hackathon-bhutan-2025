🌟 Executive Summary
The Bhutan eResidency Platform is a groundbreaking digital infrastructure that enables entrepreneurs worldwide to establish legitimate businesses in Bhutan entirely online. By combining cutting-edge technology with Bhutan's progressive digital policies, we're creating a new paradigm for international business formation.
🎯 Mission
Democratize global business opportunities by making Bhutan the world's most accessible, efficient, and entrepreneur-friendly digital jurisdiction.

🚀 Getting Started
Prerequisites
bashNode.js 18+
npm or yarn
PostgreSQL 14+
Redis 6+
MetaMask wallet (for Web3 features)
Installation
bash# Clone the repository
git clone https://github.com/bhutan-eresidency/platform.git

# Navigate to project directory
cd platform

# Install dependencies
npm install

# Set up environment variables
cp .env.example .env
# Edit .env with your configuration

# Run database migrations
npm run migrate

# Start development server
npm run dev
